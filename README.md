# Todo List App

A simple Todo List application built to learn and practice the Laravel framework.  
This project focuses on understanding how Laravel handles routing, controllers, Blade templates, models, and database interactions.

---

## Features

- ✔ **Create, Read, Update, Delete (CRUD)** tasks  
- ✔ **Soft Deletes** — deleted tasks move to a "Memory" page instead of being permanently removed  
- ✔ **Restore deleted tasks** anytime  
- ✔ Clean and simple UI built with TailwindCSS  
- ✔ Server-side validation for form inputs  

---

## Learnings

During the development of this project, I learned:

- **Routing in Laravel** (basic routes, resource routes, and controller actions)
- **How to use Controllers** to organize application logic
- **Blade templating** for building reusable and dynamic views
- **Working with Eloquent ORM**, including:
  - Creating models  
  - Querying the database  
  - Updating and deleting data  
- **Soft deletes** and restoring data using Eloquent’s `SoftDeletes` trait  
- **Form validation** using `Request` validation and error handling  
- **Using TailwindCSS** to design clean and responsive layouts  
- **CRUD workflow** from frontend form → controller → database → back to view  
- Understanding how Laravel simplifies full-stack development
